sql注入及文件上传5

本次我来介绍堆叠注入
(堆叠注入)堆叠注入的意思就是多条语句的注入,并且这两个语句是都可以执行的。
比如说;show table;create table sad like users;这两个语句同时生效。目前这个注入方式我还没有完全在
实在中用到，等用到再说吧。
文件上传
我们有时可以通过对源代码分析,发现没有对空格进行过滤,我们可以利用burpsuite将文件后缀加一个空格,就
可以成功的绕过。若观察其源代码发现其进行了去空格的操作,这里我们可以进行文件结尾加点的方式进行绕过。
若文件限制了文件末的点和空格,我们可以在文件名后面加::$DATA这样进行绕过,其作用于上面几关类似。
