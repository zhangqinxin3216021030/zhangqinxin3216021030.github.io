php与mysql 

     MySQL 是跟 PHP 配套使用的最流行的开源数据库系统。下面我讲给大家

介绍一些我学习到的mysql语句例如：mysql> set names utf8;语句 set names utf8;
用于设定数据库编码，让中文可以正常显示。下面是表中的一些设置。
如何创建数据库
CREATE DATABASE 语句用于在 MySQL 中创建数据库。实例：$sql = “CREATE DATABASE myDB”;
创建Mysql表
CREATE TABLE MyGuests (
id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
firstname VARCHAR(30) NOT NULL,NOT NULL
lastname VARCHAR(30) NOT NULL,
email VARCHAR(50),
reg_date TIMESTAMP
)
NOT NULL - 每一行都必须含有值（不能为空），null 值是不允许的。
DEFAULT value - 设置默认值
UNSIGNED - 使用无符号数值类型，0 及正数
AUTO INCREMENT - 设置 MySQL 字段的值在新增记录时每次自动增长 1
PRIMARY KEY - 设置数据表中每条记录的唯一标识。 通常列的 PRIMARY KEY 设置
为 ID 数值，与 AUTO_INCREMENT 一起使用。
下面我讲介绍如何向mysql中插入数据
这是一些语法规则：PHP 中 SQL 查询语句必须使用引号；在 SQL 查询语句中的字符串值必须加引号
数值的值不需要引号；NULL 值不需要引号。INSERT INTO 语句通常用于向 MySQL 表添加新的记录：
INSERT INTO table_name (column1, column2, column3,…)插入列表名称（第一列，第二列，第三列—）
VALUES (value1, value2, value3,…)意思是值（值1，值2， 值3）等。插入时分别写出第一列，第二列，
第三列的值，例如$sql = “INSERT INTO MyGuests (firstname, lastname, email)
VALUES (‘John’, ‘Doe’, ‘john@example.com‘)”;要注意的是每个sql语句要用分号隔开。
从mysql中读取数据
SELECT 语句用于从数据表中读取数据：例如SELECT column_name(s) FROM table_name；
我们可以使用 * 号来读取所有数据表中的字段：SELECT * FROM table_name。
WHERE 子句用于提取满足指定标准的的记录。例子如下

$result = mysqli_query($con,”SELECT * FROM Persons ORDER BY age”);

UPDATE 语句用于更新数据库表中已存在的记录。UPDATE 语法中的 WHERE 子句。WHERE 子句规定了哪些记录需要更新。
如果您想省去 WHERE 子句，所有的记录都会被更新。
mysqli_query($con,”UPDATE Persons SET Age=36
WHERE FirstName=’Peter’ AND LastName=’Griffin’”);
DELETE FROM 语句用于从数据库表中删除记录。DELETE 语法中的 WHERE 子句。WHERE 子句规定了哪些记录需要删除。
如果您想省去 WHERE 子句，所有的记录都会被删除！
mysqli_query($con,”DELETE FROM Persons WHERE LastName=’Griffin’”);（注意：关于Griffin的一整个数据都被删除了）
